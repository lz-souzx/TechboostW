TechBoostW — Sistema de Cadastro de Produtos

Descrição:
O TechBoostW é um sistema web desenvolvido para cadastro e gerenciamento de produtos. O projeto foi construído com Java, HTML, CSS e banco de dados MySQL (modelado no MySQL Workbench).
A aplicação segue a arquitetura com Model, DAO e DTO, garantindo organização do código e separação das responsabilidades.

Tecnologias:
Java
HTML
CSS
MySQL
MySQL Workbench
Arquitetura: Model, DTO, DAO

Funcionalidades:

- Cadastro de produtos
- Listagem de produtos
- Atualização/edição de produtos
- Exclusão de produtos
- Visualização organizada dos dados
- Interface simples e funcional

Como executar:

- Clone o repositorio
- Importe o banco de dados no MySQL Workbench
- Configure a conexão com o banco no projeto Java
- Compile e execute o projeto no seu servidor/aplicação web
- Acesse a interface pelo navegador
